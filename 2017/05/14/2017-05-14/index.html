生活不易，且行且珍惜

  • 主页
  • 随笔
github weibo rss

17/05/14

一个人要自律的生活，如果只在脑袋里想想，而实际行动却相去甚远，徒增痛苦与烦恼。这内心需要经过反复挣扎，懊悔。苦恼。无益！一个人如果要做成某件事情，并长期坚持下去，除了思想上的首先要觉悟之外，还需要切实的计划，以及抵制拒绝外界各种诱惑的能力，方始可能达成目标。

人总是倾向于为自己的懒惰，被诱惑吸引，逃避责任找借口，很少有人能主动拒绝自己的懒惰，拒绝诱惑以及主动承担责任。这在我自己身上体现一览无余。就好比你眼前一块没味无比的巧克力蛋糕放在你面前，你明知道你不需要它，但你很有可能还是会去尝试吃掉它。那么在你享受这没味蛋糕的背后，你就必须承担它所带来的卡路里。诚然，要抵制近在眼前的诱惑真的很困难，要抵制自己的懒惰也很难，你总是会找出无数的借口。但是一味的为自己找借口就意味着拒绝直面人生的痛苦，拒绝成长。

近来，作息时间又极其不规律，整晚的做梦，虽然这以前就有过，但我觉得是自己思想上出了问题，我抵制不了游戏的诱惑，花了太多的时间在上面，而我理智上又觉得这样非常不好，无谓浪费许多时间，以及消耗大量精力，对视力损伤也严重，而自己却又没能抵制住。总有朋友叫一起玩，这样一边处于堕落无序的状态，而思想上却又不能给自己的放纵找到足够多的理由，承认这样其实无所谓。潜意识里的自己给自己的精神压力过大导致的。但这样又会更加刺激我去放纵，周而复始。。。

2017 年 05月 14日
